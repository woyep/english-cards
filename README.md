# English Cards

A simple web app to help kids learn Cambridge English words through interactive cards.

## Demo

Open the app locally by opening `index.html` in your browser.

## Features

- **READ mode**: Look at the picture and guess the word. Click "SHOW" to reveal the answer. Click "NEXT" to move to the next card.
- **SPELL mode**: Type the word and click "CHECK" to verify your answer.

## Tech Stack

- HTML
- CSS
- JavaScript

## How to Use

1. Open `index.html`
2. Choose a category
3. Select a mode (READ / SPELL)
4. Start learning
